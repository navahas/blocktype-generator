# Rust BlockType

A simple CLI tool to print words in large ASCII/block-style text.

#### Using compiled binary: 
```bash
./rust-type <word>
```

#### With Cargo:

```bash
cargo run -- <word>```

The font is embedded in the binary — no setup required.
